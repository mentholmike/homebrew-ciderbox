# mentholmike/homebrew-ciderbox

Homebrew tap for [ciderbox](https://github.com/mentholmike/ciderbox) — Apple-native throwaway dev environments using container CLI.

## Install

```bash
brew tap mentholmike/ciderbox
brew install ciderbox
```

Or in one line:

```bash
brew install mentholmike/ciderbox/ciderbox
```

## Quickstart

```bash
ciderbox init
ciderbox orchard plant
```

## Upgrade

```bash
brew upgrade ciderbox
```
